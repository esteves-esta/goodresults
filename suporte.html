Dúvidas frequentes

Como são realizados os sites?

Nós desenvolvemos sites utilizaNdo HTML5, CSS3 e JavaScript. Tentamos sempre nós manter atualizados sobre as tendências de web, sobre a responsividade dos sites e sua usabilidade.

Porque é necessária análise de sistemas?

A fase de análise de sistemas é importante para entender quais são as funções do sistema e oque precisa ser desenvolvido. E atráves dela que podemos começar um projeto de forma clara entendendo oque será necessário ser produzido sem erro ou atrasos.

A linguagem SQL é a mesma em todos os bancos de dados?

Não. O SQL foi padronizado pela ANSI, contudo as empresas de banco de dados tem autonomia sobre seus bancos e uma versão adaptada para cada banco é comum encontramos. Mas boa parte dos comandos funcionam da mesma forma.

Como é realizado o desenvolvimento de softwares?

Nós realizamos nossos sistemas utilizando a linguagem C#.

Quais são os preços pelos serviços da GoodResults?

Dependem muito de quais serviços seram realizados, a escala do sistema, entre outros fatores. Aconselhamos a entrar em contato e pedir um orçamento. Como sabemos que é importante ter pelo menos uma noção de valor nossa equipe comercial responde logo após o primeiro contato com uma ideia de valor. Para isso precisamos que você nos explique seu projeto ou necessidade de forma sucinta e objetiva.

O QUE EU RECEBO NO FINAL DO PROJETO?

O código fonte: no final o software é totalmente seu, sem surpresas, não retemos clientes através da propriedade dos códigos fontes, nós os mantemos pelo relacionamento e experiência durante o processo de desenvolvimento;

  • Instalação no ambiente indicado pelo cliente;
  • Treinamento no ato de entrega do sistema;
  • Um período de testes onde o cliente pode indicar eventuais erros, que serão corrigidos sem custo;
  • Apresentações públicas ou internas do aplicativo ou sistema, quando contratado.

QUAIS OPÇÕES EU TENHO PARA PAGAR O PROJETO?

Trabalhamos sempre para sermos flexíveis como o mercado exige. Os projetos desenvolvidos pela GoodResults podem ser parcelados em prestações condizentes com o valor e tempo de desenvolvimento e também possuímos opções interessantes para pagamentos mais rápidos.

COMO EU ACOMPANHO O DESENVOLVIMENTO?

Durante todo o processo de desenvolvimento do seu sistema são feitas reuniões, presenciais ou remotas, com objetivo de manter a expectativa do cliente sempre alinhada ao que é produzido e entregue pela GoodResults.

Para garantir a sua imersão no novo sistema desenvolvido, dividimos tudo em pequenas entregas, de forma que é possível realizar testes e validações destas partes durante a execução do projeto.

A GoodResults FAZ MANUTENÇÃO EM SISTEMAS QUE EU JÁ TENHO?

Sim. Dependendo de uma análise técnica para ver se é possível ajustar ou evoluir o sistema da forma como o cliente necessita. Nos casos em que o sistema não atenda, nós aproveitamos o entendimento do sistema anterior e podemos propor um novo na tecnologia mais adequada ou recente.
